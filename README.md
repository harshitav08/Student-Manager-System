# Student Manager System

A simple student management system built with:

- HTML/CSS/JavaScript (frontend)
- Core Java OOP backend (service class)
- MySQL schema (database design)

## Features
- Add student
- View all students
- Search by roll number
- Remove student

## Tech Highlights
- Used localStorage for frontend demo
- Implemented CRUD logic in Java
- Designed SQL schema for real database implementation
- Demonstrated OOP: classes, objects, lists, methods

## How to run
1. Open `frontend/index.html` in your browser.
2. Use `add_student.html` to add a student.
3. View all students on `view_students.html`.
